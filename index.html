FAQ

Frequently Asked Questions

Is this really free?
Yes — 100% free with no hidden costs, no sign-up, and no daily limits. All tools work completely in your browser without uploading your files anywhere.
Are my files safe?
Absolutely. Your files never leave your device. All processing happens locally using JavaScript — nothing is sent to any server.
Does it work offline?
Yes! Install Free PDF Flash as a Progressive Web App (PWA) and all tools work even without an internet connection.
What file formats are supported?
For image-to-PDF conversion, we support JPG, JPEG, PNG, WEBP, GIF, and BMP. For PDF tools, standard PDF files are supported.
Is there a file size limit?
There's no hard limit — it depends on your device's memory. Modern devices can handle files up to several hundred MB easily.
Can I use it on my mobile phone?
Yes! The interface is designed mobile-first with touch-friendly elements. It works perfectly on Android and iOS devices.
